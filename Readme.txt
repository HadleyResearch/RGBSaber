Git repository for the RGB Lightsaber project. The RGBSaber project is an open-source electronics system for professional lightsaber replicas which provides the user with the ability to change the color of the lightsaber blade to any color the user chooses. Included in this repository are relevant documentation, source code, and pcb layout files necessary to recreate the RGBSaber project. These files are released to the general public free of charge under a Creative Commons BY-SA-3.0 license (http://creativecommons.org/licenses/by-sa/3.0/). If you do use these files in one of your projects, please let us know, as we would love to feature your project on our website.
