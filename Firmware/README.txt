Current Location: RGBSaber/Firmware

Versions: Version 1.0 of the RGBSaber firmware is legacy, assembly-based code, and
    no longer actively developed. Users and developers are highly encouraged to use
    the C-device firmware, located in v2.0/C

Software:
    Microcontroller firmware was developed using the Microchip MPLAB X IDE. A
freeware copy of this software, as well as additional information, can be found
at http://microchip.com/mplab. Firmware is written in the C programming language
and compiled using Microchip's C18 compiler. For a freeware version of this
compiler, as well as additional information, see http://www.microchip.com/c18
